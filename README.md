## Descripción

Este repositorio contiene las actividades prácticas correspondientes a la materia. Los ejercicios están organizados en notebooks y archivos de apoyo; cada actividad debe incluir código ejecutable, resultados y una breve documentación que explique los pasos realizados.

## Consignas

- **Objetivo general:** Resolver las consignas prácticas aplicando técnicas de IA vistas en clase (preprocesamiento, modelado, evaluación y documentación).
- **Formato de entrega:** Notebooks ejecutables (Jupyter / Deepnote / Colab) que incluyan código, gráficos y conclusiones.
- **Requisitos mínimos por actividad:**
  - Código reproducible que ejecute sin errores.
  - Celdas con resultados (tablas, métricas, gráficos) visibles.
  - Sección de análisis y conclusiones.
  - Archivo README con instrucciones para reproducir (dependencias y comandos).
- **Criterios de evaluación:** corrección, claridad del código, calidad del informe, reproducibilidad y discusión de resultados.
